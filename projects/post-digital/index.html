POST-DIGITAL COLLAPSE

in → December 2023
for → Programming Images
with → Travess Smalley

A series of generative animations created with Photoshop scripts, inspired by weather forecast maps.

1. Using JavaScript, a random vector shape is drawn on a randomly sized art board.
2. The layer is duplicated over and over and distortions are applied to each subsequent iteration.
3. Finally, each layer is compiled into a frame animation.
